# Clinical Appointment No-Show Prediction

## Overview
This project predicts the likelihood of patient appointment no-shows using classical machine learning techniques.

## Models Used
- Logistic Regression
- Decision Tree (Tuned with GridSearchCV)
- Neural Network (MLP)

## Features Engineered
- Lead Time
- Appointment Day of Week
- Weekend Indicator

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC
